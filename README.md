# 一、项目介绍
### 这是一个简单的秒杀系统，运用技术：Spring+SpringMvc+Mybatis+Maven+Redis+Mysql
-----------------
# 二、功能介绍：
* #### 展示秒杀商品列表 
* #### 展示商品秒杀详情页 
* #### 手机号登陆
* #### 秒杀未开始/开始提示
* #### 执行秒杀
